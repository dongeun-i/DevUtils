# 광고 수익형 유틸리티 툴 사이트 프로젝트

## 📌 프로젝트 개요
개발자 포트폴리오 겸 광고 수익 창출을 목적으로 하는 유틸리티 툴 모음 웹사이트

### 핵심 컨셉
"귀찮지만 필요한 것들을 간단하게" - 계산기, 변환기, 생성기 등 실용적인 도구들을 모아놓은 올인원 플랫폼

## 🎯 프로젝트 목표

### 1차 목표 (단기)
- Google AdSense, 카카오 애드핏 승인 및 광고 수익 창출
- 검색 트래픽 확보 (SEO 최적화)
- 포트폴리오 활용 가능한 퀄리티 확보

### 2차 목표 (중장기)
- 안정적인 월 광고 수익 달성
- 개발자 커뮤니티 내 인지도 확보
- 외주/프리랜서 포트폴리오로 활용

## 💡 주요 기능 아이디어

### 개발자 도구
- [ ] 정규식 테스터
- [ ] JSON 포맷터/검증기
- [ ] Base64 인코더/디코더
- [ ] 더미 데이터 생성기 (JSON, CSV, SQL)
- [ ] Git 커맨드 생성기
- [ ] Color Picker & Converter

### 실용 계산기
- [ ] 급여 실수령액 계산기
- [ ] 대출 이자 계산기
- [ ] 환율 변환기
- [ ] 복리 계산기
- [ ] 단위 변환기 (길이, 무게, 온도 등)

### 콘텐츠/생성 도구
- [ ] AI 문구 생성기
- [ ] 랜덤 이름/주소 생성기
- [ ] 비밀번호 생성기
- [ ] QR 코드 생성기
- [ ] 이미지 리사이저/압축기

## 🛠 기술 스택

### Frontend
- **Framework**: Next.js (SSR/SSG for SEO)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: React Context / Zustand
- **UI Components**: shadcn/ui

### Backend (필요시)
- **Runtime**: Node.js
- **Framework**: Next.js API Routes
- **Database**: PostgreSQL / MongoDB (사용자 데이터 최소화)

### DevOps & Infra
- **Hosting**: Vercel / Netlify (무료 시작)
- **Analytics**: Google Analytics 4
- **SEO**: Next.js SEO, Sitemap 자동 생성
- **Ad Platform**: Google AdSense, 카카오 애드핏

## 📊 수익화 전략

### 광고 수익
1. **Google AdSense**
   - 승인 조건: 최소 20-30개 고품질 콘텐츠 페이지
   - 전략: 각 도구별 독립 페이지 + 사용 가이드
   
2. **카카오 애드핏**
   - 한국 사용자 대상 광고 최적화
   - AdSense 보완용

### 트래픽 확보 전략
- **SEO 최적화**: 롱테일 키워드 타겟 ("OO 계산기", "OO 변환기")
- **콘텐츠 마케팅**: 각 도구 사용법 블로그 포스팅
- **개발자 커뮤니티**: Reddit, 디스콰이엇, 오픈카톡 등 공유
- **SNS**: 유용한 도구 소개 (트위터, 링크드인)

## 📝 참고 사항

### 시간 투자 계획
- 평일 퇴근 후: 1-2시간
- 주말: 4-6시간
- 총 주당 10-15시간 목표

### 우선순위
1. **아이디어 검증**: 검색량 높은 키워드 리서치
2. **MVP 개발**: 빠르게 런칭 후 피드백 수집
3. **SEO 최적화**: 트래픽 확보가 수익의 핵심
4. **광고 승인**: 고품질 콘텐츠 중심

### 주의사항
- 광고 정책 준수 (클릭 유도 금지, 콘텐츠 품질 유지)
- 사용자 프라이버시 보호 (개인정보 최소 수집)
- 모바일 최적화 (트래픽의 70% 이상 모바일)
- Core Web Vitals 최적화 (SEO 랭킹 요소)
